AI-powered business analytics and intelligence platform built with modern web technologies.

## Overview

InsightFlow is a professional SaaS analytics dashboard that provides real-time business intelligence, customer health monitoring, revenue analytics, and AI-powered insights. Built with a clean, light-mode editorial design inspired by Stripe Dashboard and Linear.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React
- **Fonts**: Plus Jakarta Sans (headings) + Source Sans 3 (body)

## Features

- **Dashboard**: KPI overview with animated counters, revenue charts, AI insights preview, customer health table
- **Analytics**: User growth tracking, conversion funnel visualization, engagement heatmap, feature adoption metrics
- **Revenue**: MRR movement breakdown, segment analysis with donut chart, detailed monthly revenue table
- **Customers**: Full customer management with search, filters, sortable columns, expandable row details
- **Engagement**: Daily active users, session metrics, feature usage ranking, user flow visualization
- **AI Insights**: Severity-filtered actionable recommendations with impact estimates

## Design Principles

- Clean light-mode interface with surgical precision
- Data-dense layouts with zero visual waste
- Consistent color system: indigo (primary), emerald (positive), rose (negative), amber (warning)
- Responsive design with mobile-first approach
- Subtle animations for professional polish
- Accessible with proper ARIA attributes and keyboard navigation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Build

```bash
npm run build
npm start
```
